Introducción

El sistema de riego es una técnica fundamental en la agricultura, ya que permite proporcionar la cantidad necesaria de agua a los cultivos para su crecimiento y desarrollo. Sin embargo, el uso inadecuado de los sistemas de riego puede tener consecuencias negativas en la productividad agrícola, el consumo de agua y el medio ambiente.

Un sistema de riego automatizado con Arduino utiliza sensores y dispositivos electrónicos para monitorear las condiciones del suelo y el clima, y ajustar el suministro de agua de acuerdo con las necesidades de los cultivos. La automatización del riego puede mejorar significativamente la eficiencia del uso del agua en la agricultura, ya que se evita el desperdicio de agua y se asegura que los cultivos reciban la cantidad exacta de agua que necesitan para su crecimiento y desarrollo.



  Objetivos

Objetivo General

Como objetivo es crear un sistema que se acople a una planta para poder evitar el continuo riego de ella y así mismo poder tener un crecimiento adecuado automático. El objetivo principal de crear un sistema de riego automático con Arduino es automatizar el proceso de riego de las plantas y reducir la necesidad de supervisión y mano de obra humana. Con un sistema de riego automático, las plantas pueden recibir la cantidad adecuada de agua en el momento correcto sin la necesidad de regar manualmente cada planta individualmente. Además, al automatizar el riego, también puedes optimizar el uso del agua y reducir el desperdicio. Un sistema de riego automatizado puede monitorear la humedad del suelo y regar solo cuando sea necesario, en lugar de hacerlo de manera constante o en intervalos fijos



Objetivo especifico

Crear las condiciones óptimas para el cultivo eficaz de plantas aun cuando no es temporada de lluvias, teniendo en cuenta un mejor sistema de riego. Crear condiciones necesarias para el cuidado eficaz de las plantas aun así no sean en temporadas, teniendo en cuenta un sistema de riego.

  Resultados esperados

1.- Programación flexible: El sistema de riego permite establecer horarios de riego personalizados para cada punto específico, teniendo en cuenta las necesidades individuales de cada área.


2.- Detección y medición de la humedad del suelo: El sistema incluye sensores de humedad del suelo en cada punto específico para monitorear el nivel de humedad. Estos sensores proporcionan datos en tiempo real, lo que permite ajustar automáticamente la frecuencia y duración del riego según las condiciones del suelo.


3.- Zonificación: El sistema divide el área en diferentes zonas según las necesidades hídricas de las plantas o cultivos en cada punto específico. Esto permite una programación más precisa y eficiente del riego, evitando desperdiciar agua.


4.- Ajuste climático: El sistema tiene en cuenta la información meteorológica local para adaptar el riego a las condiciones climáticas. Si hay lluvias pronosticadas, el sistema puede suspender el riego para evitar un exceso de humedad en el suelo.


5.- Riego inteligente: Utilizando algoritmos y datos recopilados, el sistema ajusta automáticamente la cantidad de agua suministrada en cada punto específico. Puede utilizar métodos de riego eficientes, como riego por goteo o aspersores de bajo consumo, para maximizar la eficiencia y minimizar el desperdicio de agua.


6.- Registro y análisis de datos: El sistema registra datos sobre la cantidad de agua utilizada en cada punto específico, los niveles de humedad del suelo, las condiciones climáticas y otros parámetros relevantes. Estos datos pueden ser analizados y utilizados para optimizar el rendimiento del sistema y realizar ajustes en la programación.



  Materiales


Material Descripcion
PLATAFORMA Esta es la base o estructura principal donde se montarán todos los componentes del sistema de riego automatizado. Proporciona soporte y estabilidad para los demás elementos.
SENSORES DE HUMEDAD Son dispositivos utilizados para medir la humedad del suelo. Ayudan a determinar cuándo es necesario regar las plantas y activar el sistema de riego. En este caso, se requieren dos sensores de humedad, que se pueden insertar en diferentes áreas del suelo para obtener mediciones más precisas.
SENSOR ULTRASONICO Este sensor utiliza ondas ultrasónicas para medir la distancia entre el sensor y un objeto, en este caso, el nivel de agua en un tanque o recipiente. Permite monitorear el nivel de agua disponible y activar las bombas de agua cuando sea necesario reponerlo.
TIERRA Se refiere al suelo en el que se encuentran las plantas que serán regadas. El tipo de suelo puede influir en la frecuencia y duración del riego, así como en las mediciones de humedad realizadas por los sensores.
CAJA DE ACRILICO Es un recipiente transparente fabricado en acrílico que se utiliza para alojar los componentes electrónicos y protegerlos de factores externos, como el polvo o la humedad.
MANGUERAS Son tubos flexibles que se utilizan para transportar el agua desde las bombas hasta las áreas de riego. Las mangueras deben ser lo suficientemente resistentes y de tamaño adecuado para asegurar un flujo constante de agua hacia las plantas.
BOMBAS DE AGUA 12V Son dispositivos que se encargan de impulsar el agua desde la fuente de suministro hasta las mangueras de riego. En este caso, se necesitan dos bombas de agua de 12 voltios para asegurar un flujo adecuado hacia las plantas.
MODULO RELÉ Es un componente electrónico que se utiliza para controlar el encendido y apagado de las bombas de agua. Actúa como un interruptor controlado por señales eléctricas, permitiendo que el sistema de riego se active o desactive según las condiciones de humedad detectadas por los sensores.
FUENTES DE ALIMENTACION Son dispositivos que proporcionan la energía eléctrica necesaria para alimentar los componentes electrónicos del sistema, como los sensores, las bombas y los módulos relé. Se requieren dos fuentes de alimentación, una para cada módulo relé y sus respectivos componentes asociados.
PAQUETE DE JUMPERS Son cables utilizados para establecer conexiones eléctricas entre los diferentes componentes del sistema. Permiten transmitir señales y datos entre los sensores, módulos relé y fuentes de alimentación de manera eficiente.
COMPOSTA La composta en un sistema de riego proporciona varios beneficios: retiene agua en el suelo, mejora la estructura del suelo, suministra nutrientes a las plantas y estimula la actividad microbiana. Esto resulta en un mejor crecimiento y desarrollo de las plantas, reducción en el consumo de agua y fertilizantes químicos, y un suelo más saludable en general.
ARDUINO UNO El Arduino Uno es una placa de microcontrolador de código abierto basado en el microchip ATmega328P y desarrollado por Arduino.cc. ​​ La placa está equipada con conjuntos de pines de E/S digitales y analógicas que pueden conectarse a varias placas de expansión y otros circuitos.
